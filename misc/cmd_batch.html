Run CESM Batch

Run Multiple Instances of CESM Pipeline

usage: python scripts/run_batch.py [-h] [-case CASE]
                                   [-anomaly_lat_min ANOMALY_LAT_MIN]
                                   [-anomaly_lat_max ANOMALY_LAT_MAX]
                                   [-anomaly_lon ANOMALY_LON]
                                   [-anomaly_lat_increment ANOMALY_LAT_INCREMENT]
                                   [-max_anomaly MAX_ANOMALY]
                                   [-anomaly_radius_squared ANOMALY_RADIUS_SQUARED]
                                   [-anomaly_type ANOMALY_TYPE] [-sims]
                                   -config CONFIG

Named Arguments

-case

Default: “aqua_anomaly”

-anomaly_lat_min

Default: 10.0

-anomaly_lat_max

Default: 50.0

-anomaly_lon

Default: 180.0

-anomaly_lat_increment

Default: 5.0

-max_anomaly

Default: 5.0

-anomaly_radius_squared

Default: 20.0

-anomaly_type

Default: “zonal_band”

-sims

Default: False

-config
